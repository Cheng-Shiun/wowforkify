#Project Introduce

This is a modern Single Page Application (SPA) developed as a practice project from the Udemy course: The Complete JavaScript Course 2025: From Zero to Expert! The goal of this project is to build a real-world recipe application while mastering advanced ES6+ JavaScript and the MVC (Model-View-Controller) architecture.

##Live DEMO

Deployment: Hosted on Netlify with automated CI/CD.

Link: https://wowforkify.netlify.app/

Build Tool: Bundled using Parcel to manage modules and assets.

##Features

- Search recipe: You can input some recipe name for search.
- Change servings: You can use plus or minus button to change recipe serving.
- Bookmark recipes: Every recipe you love can be bookmarked.
- Upload your own: Use Add recipe on navigation and fill your recipe data.
