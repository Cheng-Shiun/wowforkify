## Features

- **Search Recipes**: Input some recipe name for search from the Forkify API.
- **Change Servings**: Use plus or minus button to change recipe serving; quantities update dynamically.
- **Bookmark Recipes**: Every recipe you love can be bookmarked (stored via **LocalStorage**).
- **Upload Your Own**: Use "Add recipe" on navigation and fill your recipe data to sync with the API.



## Tech Highlights

- **Async/Await**: Handling complex API data fetching and error states.
- **DOM Diffing Algorithm**: Optimized UI rendering by only updating changed elements.
- **Publisher-Subscriber Pattern**: Decoupling the View from the Controller logic.
